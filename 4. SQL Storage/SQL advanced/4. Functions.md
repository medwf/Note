# What is functions ?
- you can create a function to use it later.
``` SQL
-- example
DELIMITER ##
CREATE FUNCTION SafeDiv(a INT, b INT)
RETURNS FLOAT DETERMINISTIC
BEGIN
    DECLARE result FLOAT DEFAULT 0;

    IF b != 0 THEN
        SET result = a / b;
    END IF;
    RETURN result;
END ##
DELIMITER ;
```
